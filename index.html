  8. First exam:
    • Basados en el Jupyter Notebook del Workshop 04 y teniendo como referencia de limpieza de datos el Workshop 03, realizar las siguientes tareas:

      1. Usando como plantilla el Workshop 04, crear un nuevo Notebook de Jupyter. Guardar y nombrarlo. Nombrelo como: AI_Exam_01_SUNOMBREyAPELLIDO (Si hay dos integrantes, entonces dos nombre), la extención es la de un Jupyter Notebook.

      2. Actualizar la información a la fecha (La del Notebook es de 2015 y ya hay info más reciente), tanto para los casos de tuberculosis como la población total
        URL de World Health Organization: By category > Tuberculosis > Cases and deaths
        En está página puedes conseguir la información. Igualmente en los siguientes enlaces:
        • Enlace para la info en forma de tabla: table.xml, se puede abrir con excel para observar su contenido.
        • Enlace para la info en forma de datos: data.xml, se puede abrir con excel para observar su contenido.
        Cuando descargues la información para revisión puedes contrastarla con la de los siguientes enlaces:
        • Tabla: table.xml, se puede abrir con excel para observar su contenido.
        • Datos: data.xml, se puede abrir con excel para observar su contenido.
        La información debe limpiarse y transformarse para llegar a los datos de pais, cantidad de muertes(sin VIH), población.
        • La información de nombre de país debe ser la correspondiente a nombre de paises, no de regiones (p.e. Latinamerica)
        • La información de cantidad de muertes debe ser un valor entero
        • La información de población debe ser un valor entero y deducido a partir de las muertes por 100000 habitantes y la cantidad de muertes (Deducir el cálculo algebraicamente)
        • Cualquier limpieza y transformación se debe realizar en los dataset cargados por pandas
        • Cualquier limpieza y transformación se debe realizar usando Python (Si son necesarias librerías importarlas)

      3. Conforme está en Workshop 04, escribir realizar los cálculos y poniendo los comentarios en cada una de las celdas del Notebook:
        • Para las celdas de comentarios usar el lenguaje Markdown. El idioma puede ser Ingles o Español.
        • Para las celdas de cálculos usar el lenguaje Python.
        • Es posible que algunos de los comando de Python no funcionen exactamente como está en la plantilla. Introduzca los ajustes necesarios.

      4. Recuerda siempre hacer buenas conclusiones de los datos observados.

      5. Crea una celda adicional en el Notebook donde grabas en disco el dataset resultado usando al librería pandas.

      6. Asegurate de tener en el mismo directorio el archivo con los datos obtenidos desde World Health Organization. (Tal como está en Workshop 04
